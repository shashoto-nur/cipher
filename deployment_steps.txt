
Create a project
    Go to https://firebase.google.com/
    Add project ( + )
    Click Hosting and Get Started respectively
Get firebase cli
    Run: npm install -g firebase-tools
        firebase -V
        firebase login
Initialize
    Run: firebase init
        select Hosting & press Space
        select Use an existing project
Test
    Run: firebase serve
        ctrl + c
Deploy
    Run: firebase deploy
